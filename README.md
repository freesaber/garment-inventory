# 服装进销存管理系统

一套面向服装零售行业的进销存管理系统，支持多店铺调货、库存管理、销售分析等功能。

## 核心功能

### 📦 库存管理
- 商品信息管理（SKU、颜色、尺码、条码）
- 库存盘点与调整
- 库存预警（低库存、滞销提醒）
- 批次管理与保质期跟踪

### 🏪 多店铺管理
- 多门店独立核算
- 店铺间调货申请与审批
- 调货进度跟踪
- 店铺业绩统计

### 🛒 零售管理
- POS 收银台
- 会员管理与积分
- 促销活动管理
- 销售小票打印

### 📊 报表分析
- 销售日报/月报/年报
- 商品销售排行
- 利润分析
- 库存周转率分析

### 👥 系统管理
- 用户权限管理
- 操作日志
- 数据备份与恢复
